Вебзастосунок для розшифровки автомобільних VIN-кодів.

## Посилання
https://vin-decoder-topaz.vercel.app/

## Функціонал
- Розшифровка VIN-коду через API NHTSA
- Історія трьох останніх запитів
- Валідація введеного VIN-коду
- Сторінка з переліком усіх змінних
- Детальна сторінка кожної змінної

## Локальний запуск
```bash
git clone https://github.com/твій-юзернейм/назва-репо
cd назва-репо
bun install
bun run dev
