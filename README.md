Вебзастосунок для розшифровки автомобільних VIN-кодів.

## Посилання
https://vin-decoder-topaz.vercel.app/

## Функціонал
- Розшифровка VIN-коду через API NHTSA
- Історія трьох останніх запитів
- Валідація введеного VIN-коду
- Сторінка з переліком усіх змінних
- Детальна сторінка кожної змінної
- адаптовано до мобільних пристроїв.

## Локальний запуск
```bash
git clone https://github.com/paranoiabn/VIN_Decoder
cd VIN_Decoder
bun install
bun run dev
```

$$ Також було додано наступний функціонал: збереження історії запитів у localStorage, 
натискання клавіші Enter рівнозначне натисканню кнопки "Розшифрувати", 
при цьому сторінка не перезавантажується. 
При відкритті сторінки фокус автоматично встановлюється на поле введення VIN-коду.
